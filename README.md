## Project Overview
The Smart Presentation System Using Hand Gestures is an AI-based system that allows users to control PowerPoint presentations using hand gestures instead of a mouse or keyboard. This system uses computer vision techniques to detect hand movements through a webcam and perform actions such as moving to the next slide, going back to the previous slide, or closing the presentation.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- CVZone
- Computer Vision

## Features
- Control slides using hand gestures
- Move to next slide
- Move to previous slide
- Close presentation using gesture
- Real-time hand tracking using webcam

## System Requirements
- Python 3.8 or above
- Windows Operating System
- Webcam
